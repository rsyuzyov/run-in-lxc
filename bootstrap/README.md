# Bootstrap — базовая настройка LXC-контейнера

Скрипт `setup.sh` выполняет первоначальную настройку контейнера после его создания.

## Использование

```bash
./setup.sh [опции]
```

### Параметры

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `-tz, --timezone ZONE` | Часовой пояс | `Europe/Moscow` (GMT+3) |
| `-h, --help` | Показать справку | — |

### Примеры

```bash
# По умолчанию — GMT+3 (Europe/Moscow)
./setup.sh

# Указать другой часовой пояс
./setup.sh --timezone Europe/Kiev
./setup.sh -tz UTC
./setup.sh -tz America/New_York
```

## Что делает скрипт

1. **Настройка локалей**
   - Устанавливает `ru_RU.UTF-8` и `en_US.UTF-8`
   - Русская локаль по умолчанию

2. **Настройка часового пояса**
   - По умолчанию GMT+3 (Europe/Moscow)
   - Можно изменить через параметр `--timezone`

3. **Обновление системы**
   - `apt update` + `apt dist-upgrade`

4. **Установка базовых утилит**
   - `mc` — файловый менеджер
   - `atop` — мониторинг системы
   - `htop` — интерактивный просмотр процессов
   - `curl`, `wget` — загрузка файлов
   - `git` — система контроля версий
   - `openssh-server` — SSH-сервер

5. **Настройка SSH**
   - Разрешает вход root по паролю
   - Включает парольную аутентификацию

6. **Очистка**
   - Удаляет неиспользуемые пакеты
   - Очищает кэш apt

## Требования

- Debian/Ubuntu-based LXC контейнер
- Запуск от root
